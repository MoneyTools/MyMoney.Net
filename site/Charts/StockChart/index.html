Stock Chart#

When you select the "View by Security" view you will get an additional tab labelled "Stock" which will show the complete stock price history for that security.  This will only happen if you have setup your Stock Quote Services.

Note: This graph takes stock splits into account, so if you see a graph with a vertical decline like this:

Then chances are you are missing Stock Split information on that date.  See Securities on how to provide stock split information.

Note this is a type of Trend Graph so the same context menu items are available here.
