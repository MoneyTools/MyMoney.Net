Options#

Under the View menu there is an Optionsâ€¦ item that will bring up the following options panel:

You can use this to:

  • Change your database password (only visible if passwords are supported on your database file type).
  • Select a new UI Theme (Light or Dark).
  • Change when the fiscal year starts for reports and charts.
  • Default currency for reports
  • Whether to show currency info in the Accounts View.
  • Range of dates to search for matching automatic F12 transfers.
  • Enable/Disable the Rental Management feature
  • Enable/Disable playing of any sounds.
  • Configure how Ctrl+Space works during account balancing.
  • Enable loading of .ofx files using UTF-8 encoding.
